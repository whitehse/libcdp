# ADR-005: Embedded Arrays in Event Structs

**Status:** Accepted
**Date:** 2025-01-01

## Context

Events contain variable-length data (JSON strings, method names, etc.).
We could dynamically allocate strings or use fixed-size embedded arrays.

## Decision

Use embedded fixed-size arrays in event structs. Maximum sizes are
defined by CDP_MAX_* constants. This avoids heap allocation in the
event path and makes events trivially copyable.

## Consequences

- **Positive:** No malloc/free in event path
- **Positive:** Events are trivially copyable
- **Positive:** Cache-friendly layout
- **Negative:** Fixed maximum sizes may truncate large messages
- **Negative:** Wastes memory for small messages
