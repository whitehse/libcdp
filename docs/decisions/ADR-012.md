# ADR-012: C11 Standard

**Status:** Accepted
**Date:** 2025-01-01

## Context

The library needs to choose a C standard for compatibility and feature
availability.

## Decision

Target the C11 standard. This provides:
- `_Static_assert` for compile-time checks
- Anonymous structs and unions (used in cdp_event_t)
- `alignas` and `alignof`
- `_Noreturn` and `_Generic` if needed

Compiler flags: -Wall -Wextra -Wpedantic -Werror

## Consequences

- **Positive:** Modern features, wide compiler support
- **Positive:** Anonymous unions simplify event access
- **Positive:** Strict warnings catch bugs early
- **Negative:** Not compatible with C89/C99-only compilers
- **Negative:** Some embedded compilers may not support C11
