# ADR-007: Domain Classification by Method Prefix

**Status:** Accepted
**Date:** 2025-01-01

## Context

CDP messages have a "method" field like "Page.navigate" or
"Network.requestWillBeSent". The library needs to classify these into
domains for structured event handling.

## Decision

Extract the domain by splitting the method string on the first '.'
character. Use string prefix matching against known domain names
("Page.", "DOM.", "Runtime.", etc.).

## Consequences

- **Positive:** Simple and fast classification
- **Positive:** No lookup tables needed
- **Positive:** Easy to add new domains
- **Negative:** Unknown domains classified as OTHER
- **Negative:** Case-sensitive matching
